## 🚀 Features
- *Real-Time Tracking:* Watch the collector location update live on a map.
- *Dynamic User Profiles:* Secure Login/Signup with real-time stats and points.
- *Collector Dashboard:* Dedicated UI for collectors to manage waste pickups and history.
- *AI Warehouse Sorting:* Integration ready.

## 🛠️ Tech Stack
- *Frontend:* HTML, Vanilla CSS (Mobile-First Responsive Design), JavaScript
- *Backend:* Node.js, Express.js, Socket.IO
- *Database:* MongoDB Atlas
- *Hosting:* Render.com
